# Instrucciones para Codex

Este proyecto es un ejercicio educativo de Java puro sobre Programación Orientada a Objetos.

## Objetivo

Implementar un sistema básico para una biblioteca aplicando:
- Encapsulamiento
- Herencia
- Polimorfismo
- Abstracción
- Interfaces
- Enumerados
- Organización por paquetes

## Reglas técnicas

- Usar Java puro.
- No usar Spring Boot.
- No usar frameworks externos.
- No usar base de datos.
- No usar Maven ni Gradle si el proyecto no lo requiere.
- Mantener el código simple, claro y entendible para estudiantes.
- Usar nombres en español para clases, métodos y atributos.
- Documentar las clases principales con comentarios JavaDoc.
- El proyecto debe poder ejecutarse desde una clase `Main`.

## Estructura esperada

Usar la siguiente estructura de paquetes:

```text
com.ejemplo.biblioteca
com.ejemplo.biblioteca.modelo
com.ejemplo.biblioteca.servicio
```
